Atelier

Atelier was a collaborative group project done between our Software Engineering bootcamp group with a UX team from a UX cohort. The MERN tech stack was used to recreate the high-fidelity prototype we were given via Figma.

Survivor

Survivor is a text-based story game following the adventure of the only survivor of a plane crash. Player must be able to survive and gather enough wood to create a signal fire. Tech used: HTML, CSS, JavaScript.

Johnny's Sushi

Johnny's Sushi was the third project that was done during my Software Engineering bootcamp. It is an SPA that allows the user to login, add food and beverages to cart for ordering. It was built with MERN, with a dash of authentication using JWT.
